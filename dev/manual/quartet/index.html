Reading a network

An extended Newick string can be read by the function readTopology from PhyloNetworks. Here we will use a five tip network:

(5,(4,((3,(2)#H6:::0.6),(1,#H6:::0.4))));.

julia> network = readTopology("(5,(4,((3,(2)#H6:::0.6),(1,#H6:::0.4))));")ERROR: UndefVarError: readTopology not defined

Extract quartet(s)

net=readTopology q=extractNQuartets2(net) print(q)
