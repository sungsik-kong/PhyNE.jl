Reading a network

An extended Newick string can be read by the function readTopology from PhyloNetworks. Here we will use a five tip network:

(5,(4,((3,(2)#H6:::0.6),(1,#H6:::0.4))));.

julia> using PhyNE
julia> network = readTopology("(5,(4,((3,(2)#H6:::0.6),(1,#H6:::0.4))));")PhyloNetworks.HybridNetwork, Rooted Network
11 edges
11 nodes: 5 tips, 1 hybrid nodes, 5 internal tree nodes.
tip labels: 5, 4, 3, 2, ...
(5,(4,((3,(2)#H6:::0.6),(1,#H6:::0.4))));
